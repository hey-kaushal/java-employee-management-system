# 📍 DevOps Learning Roadmap

## Phase 1 — Foundation
- [ ] Linux basics (file system, permissions, processes, networking)
- [ ] Git & GitHub (branching, PRs, webhooks)
- [ ] Maven (build lifecycle, POM, dependencies)

## Phase 2 — CI/CD
- [ ] Jenkins (pipelines, agents, shared libraries)
- [ ] SonarQube (code quality gates, integration with Jenkins)

## Phase 3 — Containerisation
- [ ] Docker (images, containers, networking, volumes)
- [ ] Docker Compose (multi-service apps)

## Phase 4 — Orchestration
- [ ] Kubernetes (Pods, Deployments, Services, Ingress, ConfigMaps, Secrets)
- [ ] Helm (chart packaging and release management)

## Phase 5 — Cloud & IaC
- [ ] AWS (EC2, S3, IAM, VPC, EKS, RDS)
- [ ] Terraform (providers, state, modules)
- [ ] Ansible (playbooks, roles, inventory)

## Phase 6 — Reverse Proxy & Monitoring
- [ ] Nginx (reverse proxy, SSL termination, load balancing)
- [ ] Prometheus + Grafana (metrics, dashboards)

## ✅ Project Milestones
- [ ] Deploy Employee Management System with full CI/CD on AWS EKS
- [ ] Set up Jenkins → SonarQube → Docker → K8s pipeline
- [ ] IaC provisioning via Terraform + Ansible
